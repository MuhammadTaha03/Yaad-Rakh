# Module 2 – Task Management (Core) Implementation Plan for **Yaad Rakh**

> **Version:** 1.0
> **Depends on:** Module 1 (Onboarding) — already complete
> **Stack:** Flutter + Hive (offline-first) + Firebase Firestore (background sync)
> **State Management:** Provider

---

## 1. Overview

Module 2 is the heart of the app. Users can add, edit, delete, and complete tasks. Tasks are always saved locally in Hive first. When internet is available, changes sync silently to Firestore in the background. Natural language parsing lets Urdu/Roman Urdu/English speakers type naturally and have the date/time auto-filled.

## 2. Task Data Model

```dart
// lib/tasks/models/task.dart

import 'package:hive/hive.dart';

part 'task.g.dart'; // generated by build_runner

enum RepeatOption { none, daily, weekly, monthly, custom }
enum TaskCategory { home, work, study, shopping, other }

@HiveType(typeId: 1)
class Task extends HiveObject {
  @HiveField(0)
  final String id;

  @HiveField(1)
  String title;

  @HiveField(2)
  DateTime? dueDate;

  @HiveField(3)
  String? dueTime; // stored as "HH:mm" string

  @HiveField(4)
  RepeatOption repeatOption;

  @HiveField(5)
  TaskCategory category;

  @HiveField(6)
  bool isCompleted;

  @HiveField(7)
  final DateTime createdAt;

  @HiveField(8)
  final String languageId; // 'en', 'ur', 'roman_ur'

  @HiveField(9)
  bool isSyncedToFirestore;

  Task({
    required this.id,
    required this.title,
    this.dueDate,
    this.dueTime,
    this.repeatOption = RepeatOption.none,
    this.category = TaskCategory.other,
    this.isCompleted = false,
    required this.createdAt,
    required this.languageId,
    this.isSyncedToFirestore = false,
  });

  Map<String, dynamic> toFirestore() => {
        'id': id,
        'title': title,
        'dueDate': dueDate?.toIso8601String(),
        'dueTime': dueTime,
        'repeatOption': repeatOption.name,
        'category': category.name,
        'isCompleted': isCompleted,
        'createdAt': createdAt.toIso8601String(),
        'languageId': languageId,
      };

  factory Task.fromFirestore(Map<String, dynamic> data) => Task(
        id: data['id'] as String,
        title: data['title'] as String,
        dueDate: data['dueDate'] != null
            ? DateTime.parse(data['dueDate'] as String)
            : null,
        dueTime: data['dueTime'] as String?,
        repeatOption: RepeatOption.values.firstWhere(
          (e) => e.name == data['repeatOption'],
          orElse: () => RepeatOption.none,
        ),
        category: TaskCategory.values.firstWhere(
          (e) => e.name == data['category'],
          orElse: () => TaskCategory.other,
        ),
        isCompleted: data['isCompleted'] as bool? ?? false,
        createdAt: DateTime.parse(data['createdAt'] as String),
        languageId: data['languageId'] as String? ?? 'en',
      );
}
```

### 4.2 AddTaskScreen

Minimal — only four visible fields:

1. `TextField` — task title (autofocus, RTL for Urdu, max 2 lines).
2. `DateTimePicker` widget — optional due date + time.
3. `RepeatSelector` — bottom sheet with Daily / Weekly / Monthly / None.
4. Row of `CategoryChip` — Home / Work / Study / Shopping / Other.

**Natural language parsing** runs on every `onChanged` of the title field and auto-fills date/time if a pattern is detected (see Section 8).

Save button calls `provider.addTask(task)`.

### 4.3 EditTaskScreen

Same layout as `AddTaskScreen`, pre-filled with existing task values.

Save button calls `provider.updateTask(task)`.

## 6. TaskProvider — Full Implementation

```dart
// lib/tasks/task_provider.dart

import 'package:flutter/material.dart';
import 'package:hive/hive.dart';
import 'package:uuid/uuid.dart';
import 'models/task.dart';
import 'services/firestore_sync_service.dart';

class TaskProvider extends ChangeNotifier {
  final Box<Task> _box = Hive.box<Task>('tasks');
  final FirestoreSyncService _sync = FirestoreSyncService();
  final _uuid = const Uuid();

  List<Task> get tasks => _box.values.toList()
    ..sort((a, b) => (a.dueDate ?? DateTime(2100))
        .compareTo(b.dueDate ?? DateTime(2100)));

  List<Task> get pendingTasks =>
      tasks.where((t) => !t.isCompleted).toList();

  List<Task> get completedTasks =>
      tasks.where((t) => t.isCompleted).toList();

  Future<void> addTask(Task task) async {
    await _box.put(task.id, task);
    notifyListeners();
    _sync.pushTask(task); // fire-and-forget background sync
  }

  Future<void> updateTask(Task task) async {
    await _box.put(task.id, task);
    notifyListeners();
    _sync.pushTask(task);
  }

  Future<void> toggleComplete(String id) async {
    final task = _box.get(id);
    if (task == null) return;
    task.isCompleted = !task.isCompleted;
    await task.save();
    notifyListeners();
    _sync.pushTask(task);
    if (task.isCompleted) _generateNextRecurring(task);
  }

  // Returns the deleted task so the caller can show Undo
  Future<Task?> deleteTask(String id) async {
    final task = _box.get(id);
    if (task == null) return null;
    await _box.delete(id);
    notifyListeners();
    _sync.deleteTask(id);
    return task;
  }

  Future<void> undoDelete(Task task) async {
    await _box.put(task.id, task);
    notifyListeners();
    _sync.pushTask(task);
  }

  Future<void> syncFromFirestore(String uid) async {
    final remoteTasks = await _sync.fetchAll(uid);
    for (final t in remoteTasks) {
      if (!_box.containsKey(t.id)) {
        await _box.put(t.id, t);
      }
    }
    notifyListeners();
  }

  // Generates next occurrence for recurring tasks
  void _generateNextRecurring(Task completed) {
    if (completed.repeatOption == RepeatOption.none) return;
    if (completed.dueDate == null) return;

    DateTime nextDate;
    switch (completed.repeatOption) {
      case RepeatOption.daily:
        nextDate = completed.dueDate!.add(const Duration(days: 1));
        break;
      case RepeatOption.weekly:
        nextDate = completed.dueDate!.add(const Duration(days: 7));
        break;
      case RepeatOption.monthly:
        nextDate = DateTime(
          completed.dueDate!.year,
          completed.dueDate!.month + 1,
          completed.dueDate!.day,
        );
        break;
      default:
        return;
    }

    final next = Task(
      id: _uuid.v4(),
      title: completed.title,
      dueDate: nextDate,
      dueTime: completed.dueTime,
      repeatOption: completed.repeatOption,
      category: completed.category,
      createdAt: DateTime.now(),
      languageId: completed.languageId,
    );
    addTask(next);
  }

  String newId() => _uuid.v4();
}
```

## 8. Natural Language Parser (Rule-Based)

```dart
// lib/tasks/services/natural_language_parser.dart

class ParsedTaskInput {
  final String cleanTitle;
  final DateTime? detectedDate;
  final String? detectedTime; // "HH:mm"

  const ParsedTaskInput({
    required this.cleanTitle,
    this.detectedDate,
    this.detectedTime,
  });
}

class NaturalLanguageParser {
  static ParsedTaskInput parse(String input) {
    String text = input.toLowerCase().trim();
    DateTime? date;
    String? time;

    final now = DateTime.now();
    final today = DateTime(now.year, now.month, now.day);

    // --- Date keywords ---
    if (_contains(text, ['aaj', 'آج', 'today'])) {
      date = today;
    } else if (_contains(text, ['kal', 'کل', 'tomorrow'])) {
      date = today.add(const Duration(days: 1));
    } else if (_contains(text, ['parso', 'پرسوں', 'day after'])) {
      date = today.add(const Duration(days: 2));
    } else if (_contains(text, ['is hafte', 'اس ہفتے', 'this week'])) {
      date = today.add(const Duration(days: 3));
    } else if (_contains(text, ['agale hafte', 'اگلے ہفتے', 'next week'])) {
      date = today.add(const Duration(days: 7));
    }

    // Day names (English + Roman Urdu + Urdu)
    const dayNames = {
      'monday': 1, 'somwar': 1, 'سوموار': 1,
      'tuesday': 2, 'mangal': 2, 'منگل': 2,
      'wednesday': 3, 'budh': 3, 'بدھ': 3,
      'thursday': 4, 'jumerat': 4, 'جمعرات': 4,
      'friday': 5, 'jumma': 5, 'جمعہ': 5,
      'saturday': 6, 'hafta': 6, 'ہفتہ': 6,
      'sunday': 7, 'itwar': 7, 'اتوار': 7,
    };
    for (final entry in dayNames.entries) {
      if (text.contains(entry.key)) {
        final targetWeekday = entry.value;
        int diff = targetWeekday - now.weekday;
        if (diff <= 0) diff += 7;
        date = today.add(Duration(days: diff));
        break;
      }
    }

    // --- Time patterns ---
    // Matches: "9 baje", "9:30", "9 AM", "9 PM", "9 bajay"
    final timeRegex = RegExp(
      r'(\d{1,2})(?::(\d{2}))?\s*'
      r'(baje|bajay|بجے|am|pm|a\.m|p\.m)?',
      caseSensitive: false,
    );
    final match = timeRegex.firstMatch(text);
    if (match != null) {
      int hour = int.parse(match.group(1)!);
      int minute = int.tryParse(match.group(2) ?? '0') ?? 0;
      final suffix = (match.group(3) ?? '').toLowerCase();

      if (suffix == 'pm' || suffix == 'p.m') {
        if (hour < 12) hour += 12;
      } else if (suffix == 'am' || suffix == 'a.m') {
        if (hour == 12) hour = 0;
      }

      time = '${hour.toString().padLeft(2, '0')}:${minute.toString().padLeft(2, '0')}';
    }

    // Remove matched keywords from title
    String clean = input;
    final removeWords = [
      'aaj', 'kal', 'parso', 'baje', 'bajay', 'today', 'tomorrow',
      'is hafte', 'agale hafte', 'this week', 'next week',
    ];
    for (final w in removeWords) {
      clean = clean.replaceAll(RegExp(w, caseSensitive: false), '').trim();
    }
    clean = clean.replaceAll(timeRegex, '').trim();
    clean = clean.replaceAll(RegExp(r'\s+'), ' ').trim();

    return ParsedTaskInput(
      cleanTitle: clean.isEmpty ? input : clean,
      detectedDate: date,
      detectedTime: time,
    );
  }

  static bool _contains(String text, List<String> keywords) =>
      keywords.any((k) => text.contains(k));
}
```

```dart
// lib/tasks/widgets/task_tile.dart  — relevant animation part

class _AnimatedCheck extends StatefulWidget {
  final bool isCompleted;
  final VoidCallback onTap;
  const _AnimatedCheck({required this.isCompleted, required this.onTap});

  @override
  State<_AnimatedCheck> createState() => _AnimatedCheckState();
}

class _AnimatedCheckState extends State<_AnimatedCheck>
    with SingleTickerProviderStateMixin {
  late AnimationController _ctrl;
  late Animation<double> _scale;

  @override
  void initState() {
    super.initState();
    _ctrl = AnimationController(
      vsync: this,
      duration: const Duration(milliseconds: 300),
    );
    _scale = Tween<double>(begin: 1.0, end: 1.4).animate(
      CurvedAnimation(parent: _ctrl, curve: Curves.elasticOut),
    );
  }

  @override
  void dispose() {
    _ctrl.dispose();
    super.dispose();
  }

  void _handleTap() {
    _ctrl.forward(from: 0).then((_) => _ctrl.reverse());
    widget.onTap();
  }

  @override
  Widget build(BuildContext context) {
    return GestureDetector(
      onTap: _handleTap,
      child: ScaleTransition(
        scale: _scale,
        child: AnimatedContainer(
          duration: const Duration(milliseconds: 200),
          width: 28,
          height: 28,
          decoration: BoxDecoration(
            shape: BoxShape.circle,
            color: widget.isCompleted
                ? const Color(0xFF3B82F6)
                : Colors.transparent,
            border: Border.all(
              color: widget.isCompleted
                  ? const Color(0xFF3B82F6)
                  : Colors.grey.shade400,
              width: 2,
            ),
          ),
          child: widget.isCompleted
              ? const Icon(Icons.check, color: Colors.white, size: 16)
              : null,
        ),
      ),
    );
  }
}
```

## 10. Delete with Undo

```dart
// In TaskListScreen — delete gesture + SnackBar undo

Future<void> _deleteWithUndo(BuildContext context, String taskId) async {
  final provider = context.read<TaskProvider>();
  final deleted = await provider.deleteTask(taskId);
  if (deleted == null) return;

  if (!context.mounted) return;
  ScaffoldMessenger.of(context).showSnackBar(
    SnackBar(
      content: Text(_t(context, 'taskDeleted')),
      duration: const Duration(seconds: 4),
      action: SnackBarAction(
        label: _t(context, 'undo'),
        onPressed: () => provider.undoDelete(deleted),
      ),
    ),
  );
}
```

## 11. Firebase Firestore Sync Service

```dart
// lib/tasks/services/firestore_sync_service.dart

import 'package:cloud_firestore/cloud_firestore.dart';
import 'package:hive/hive.dart';
import '../models/task.dart';

class FirestoreSyncService {
  final _db = FirebaseFirestore.instance;

  String? _uid() => Hive.box('user').get('uid') as String?;

  CollectionReference<Map<String, dynamic>>? _col() {
    final uid = _uid();
    if (uid == null) return null;
    return _db.collection('users').doc(uid).collection('tasks');
  }

  // Push one task — silent, no UI impact
  Future<void> pushTask(Task task) async {
    final col = _col();
    if (col == null) return; // user is offline / not signed in
    try {
      await col.doc(task.id).set(task.toFirestore(), SetOptions(merge: true));
      task.isSyncedToFirestore = true;
      await task.save();
    } catch (_) {
      // Will retry on next push — offline resilience
    }
  }

  Future<void> deleteTask(String id) async {
    final col = _col();
    if (col == null) return;
    try {
      await col.doc(id).delete();
    } catch (_) {}
  }

  Future<List<Task>> fetchAll(String uid) async {
    try {
      final snap = await _db
          .collection('users')
          .doc(uid)
          .collection('tasks')
          .get();
      return snap.docs
          .map((d) => Task.fromFirestore(d.data()))
          .toList();
    } catch (_) {
      return [];
    }
  }
}
```

### Conflict Resolution Strategy

| Scenario | Resolution |
|----------|------------|
| Same task edited offline + online | **Last write wins** — `updatedAt` timestamp (add this field if needed). |
| Task deleted locally, still in Firestore | Delete wins — on next sync, local delete propagates. |
| Task exists in Firestore but not Hive | Pull from Firestore on `syncFromFirestore()` — inserts if missing. |
| User not signed in | All writes stay in Hive. On sign-in, push all unsynced tasks (`isSyncedToFirestore == false`). |

## 14. Edge Cases & Handling

| Edge Case | Mitigation |
|-----------|------------|
| User types Urdu script in task title | `TextField` with `textDirection: TextDirection.rtl` when `languageId == 'ur'` |
| Natural language parse returns wrong date | Date/time fields are editable — user can override auto-filled values |
| Hive box `'tasks'` not open | Wrap `Hive.box<Task>('tasks')` in `try/catch` — show error dialog |
| Task list empty | Show `EmptyTaskView` with a CTA "Add your first task" |
| Recurring task — month has fewer days | Use `DateTime(year, month + 1, 1).subtract(Duration(days: 1))` for last valid day |
| Firestore write fails (offline) | `isSyncedToFirestore = false` stays — retried on next `pushTask` call |
| User not signed in — Firestore sync skipped | `_uid()` returns `null` — `pushTask` returns early silently |
| Two tasks with same title | IDs are UUIDs — no conflict |
| `uuid` not imported | Add `import 'package:uuid/uuid.dart';` in `task_provider.dart` |
| Dismissible re-adds on undo | `undoDelete` uses original task object with same `id` — safe |
| `build_runner` conflict on re-run | Use `--delete-conflicting-outputs` flag |
